# AI Model Dashboard

Interactive, view-only dashboard of the AI model landscape — filter models by category, deployment tier (on-device / data-center / cloud), hardware footprint, license, and activity/domain.

**Live site:** _(GitHub Pages URL — enable Pages on the `main` branch, root folder)_ → `https://<your-user>.github.io/AI-Model-dashboard/`

This repository is public for **viewing only**. It contains a single self-contained `index.html`, mirrored automatically from a private source repository ("AI Models and tooling") whenever the dashboard is updated. Please don't send pull requests here — the source of truth (data, build scripts, full catalog) lives in the private repo. For the orignal repo it is live at https://rndu53r.github.io/AI-Model-dashboard/

_Hardware figures shown in the dashboard are heuristic estimates from parameter count; verify against model cards before provisioning._
